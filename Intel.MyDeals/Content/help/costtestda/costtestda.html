Cost Test for Division Approvers

The Cost Test Tab, located in the Manage tab as a menu item provides a quick and simple way to find cost test failures. Users with the ability to override cost test failures may view by failures only, passes, incompletes, or NA's.

The override capability is located in the Override column, when selecting "yes" a pop up with override options will be presented. You may view in the compact view or expand to see additional columns of information.

The cost test table also has the "Group Deals" column which let's you view the groupings the system has done for the cost test. If the Geo Approver selected to exclude any deals from the grouping you are able to view them here.

When all overrides have been completed proceed to the Approval View for approving deals.

TIP: You are also able to view the cost test tab by clicking on the icon next to the strategy you would like to view from the approval view in the manage tab.
