Cost Test for Super Geo Approvers

The Cost Test Tab, located in the Manage tab as a menu item provides a quick and simple way to view cost test results. Users may view by failures only, passes, incompletes, or NA's.

The cost test table also has the "Group Deals" column which let's you view the groupings the system has done for the cost test. If you would like to make exclusions navigate to the Grouping Exclusions tab

TIP: You are also able to view the cost test tab by clicking on the icon next to the strategy you would like to view from the approval screen in the manage tab.
